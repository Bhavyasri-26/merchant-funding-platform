# Merchant Funding & ACH Settlement Platform

## Overview

A production-ready payment processing platform inspired by real-world banking systems.

This project is being developed using Java Spring Boot and modern microservices architecture.

---

## Technology Stack

- Java 21
- Spring Boot 3
- Spring Cloud
- Spring Security
- Kafka
- PostgreSQL
- Docker
- Angular

---

## Planned Features

- User Authentication
- Merchant Management
- Funding Request Processing
- Payment Processing
- Bank Acknowledgement Simulation
- Kafka Event Processing
- Service Discovery (Eureka)
- API Gateway
- Load Balancing
- Circuit Breaker
- Docker Deployment
